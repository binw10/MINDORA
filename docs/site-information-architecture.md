## Page: Products / Solutions

页面定位：介绍 MINDORA 的产品定义、核心机制与行业解决方案方向。

### Module 1

- 模块标题：Mind Map as Creative Hub / 思维导图是创意中枢
- 模块说明：定义 MINDORA 是什么。
- 文案内容：

MINDORA 不是“AI + 思维导图”的简单叠加，而是以思维导图为中枢的智能创意平台。

它将灵感捕捉、结构重组、视觉具象、表达输出与行动路径连接起来，让想法从零散输入逐步转化为可呈现、可执行的成果。

### Module 2

- 模块标题：Creative Evolution Loop / 创意进化闭环
- 模块说明：说明产品如何工作，但不展开技术架构。
- 文案内容：

MINDORA 的核心机制是一套持续共创的创意进化闭环：输入碎片、形成导图骨架、延伸节点、沉淀创意结晶、完成视觉具象，并通过灵感反哺进入下一轮创作。

这不是一次性的输入输出，而是 Human + AI Co-Creation 的持续过程。

### Module 3

- 模块标题：Product Outputs / 可执行输出
- 模块说明：展示产品能连接到哪些表达结果。
- 文案内容：

MINDORA 用思维导图调度启发、结构、视觉与输出。

面向真实使用场景，平台可支持 Markdown 提纲、PPT、海报、文档与任务看板等结果导向输出。

### Module 4

- 模块标题：Vertical Solutions / 行业垂直解决方案
- 模块说明：呈现 PPT 中已覆盖的行业方向。
- 文案内容：

MINDORA 的行业解决方案覆盖教育、广告营销、咨询策略、影视多媒体与产品设计等方向。

在教育场景中，它支持知识整理、文献萃取与视觉笔记。

在广告营销中，它支持故事线、Moodboard 与创意策划。

在咨询策略中，它支持商业逻辑、竞品矩阵与提案表达。

在影视多媒体中，它支持分镜构思、视觉草图与概念海报。

在产品设计中，它支持产品线衍生、执行方案与项目路径。

## Page: Technology / AI Capabilities

页面定位：说明 MINDORA 的 AI 能力如何服务产品与创意流程；不讲模型参数或底层算法。

### Module 1

- 模块标题：Human-Led AI / 人始终主导创意
- 模块说明：建立技术表达边界。
- 文案内容：

MINDORA 的 AI 能力以增强人类思考为目标，而不是替代人类创意。

AI 是第二大脑，思维导图是中枢，人始终主导创意。

### Module 2

- 模块标题：From Fragment to Structure / 从碎片到结构
- 模块说明：说明 AI 能力在思维整理中的作用。
- 文案内容：

面对想法太多、组织不清、结构难以形成方案的问题，MINDORA 提供结构化梳理，帮助用户化繁为简，建立主干。

平台通过抽取核心主干，消除噪音，保留灵感本质并重组框架。

### Module 3

- 模块标题：From Concept to Visual / 从概念到视觉
- 模块说明：说明多模态能力的官网化表达。
- 文案内容：

MINDORA 支持在创意节点上进一步延伸，并通过多模态生图能力，让抽象概念转化为视觉草图。

视觉结果不是创作终点，而会反哺下一轮思考与表达。

### Module 4

- 模块标题：Scenario-Driven, Not Model-Driven / 场景驱动，而非模型驱动
- 模块说明：明确技术叙事方式。
- 文案内容：

MINDORA 不兜售算法与参数。

平台传播的不是“算法与参数”，而是“思维的进化”与“创意的具象化”。

技术能力最终服务于学习整理、创意策划、商业提案、团队协作等真实使用场景。

### Module 5

- 模块标题：Technology Architecture / 技术架构
- 模块说明：标注信息边界。
- 文案内容：

PPT 未覆盖具体模型架构、算法细节、系统架构或基础设施设计。

官网如需呈现技术架构，应在后续补充经确认的技术资料后再展开。

## Page: Why MINDORA / Competitive Advantages

页面定位：说明 MINDORA 的差异化价值，以及为什么它不是普通 AI 工具。

### Module 1

- 模块标题：Mind-Centered, Not Answer-Ended / 以思维为中心，不以答案为终点
- 模块说明：呈现核心产品差异。
- 文案内容：

MINDORA 以思维导图作为创意主干，保留人类真实的非线性思考路径。

平台关注思维主干、节点深挖拓宽，以及动态伸缩与进化。

### Module 2

- 模块标题：From Answers to Thinking Order / 从答案到可执行秩序
- 模块说明：回应 AI 工具的常见断点。
- 文案内容：

当“答案”越来越廉价，真正稀缺的是把想法重新组织成可执行秩序。

MINDORA 解决的不只是生成问题，而是工具碎片化、思路难以组织、创意缺乏结构化放大的问题。

### Module 3

- 模块标题：Product as Methodology / 产品即方法论
- 模块说明：说明 MINDORA 的产品优势。
- 文案内容：

MINDORA 不把产品停留在单次生成，而是用思维导图调度启发、结构、视觉与输出。

从创意节点到视觉草图，再到提纲、文档、PPT 与任务看板，平台导向可执行结果。

### Module 4

- 模块标题：Regional & Cultural Understanding / 区域与文化理解
- 模块说明：呈现马来西亚与东南亚优势。
- 文案内容：

马来西亚是天然的跨文化试验场。

多语言、多文化、多行业交汇，使跨文化创意需求快速增长。

MINDORA 立足马来西亚，辐射东南亚市场，并关注英语、中文、马来语、印尼语等区域语言体系。

### Module 5

- 模块标题：Three Understandings / 三个理解
- 模块说明：总结 Why MINDORA。
- 文案内容：

MINDORA 理解创意，而不是替代创意。

MINDORA 理解区域，而不是复制模式。

MINDORA 理解人，而不是只理解数据。
